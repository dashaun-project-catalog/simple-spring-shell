# Simple Spring Shell

Project to demonstrate the use of Spring Shell.

It has a custom shell prompt and a single command to `generate` a new file.

## Prerequisites

- Java 21
- GraalVM to generate the native CLI

## Quick Start

```bash
./mvnw spring-boot:run
```

## Generate native CLI

```bash
./mvnw -Pnative native:compile
```
